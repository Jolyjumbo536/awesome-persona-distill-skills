## 🛠️ How to Use Agent Skills

Once you install the program, you can start applying these persona skills to your own projects. Think of a persona as a set of rules for how your agent acts.

### Building Relationships
You can select different relationship modules. These modules change the tone of the agent. Use these if you want your agent to act like a mentor, a peer, or a friend. Each skill helps the agent recognize how to treat different people in a conversation.

### Managing Memory
Memory skills allow the agent to remember past interactions. You can configure the agent to save specific topics or ignore others. This keeps the agent precise and prevents it from forgetting important details about your past chats.

### Solving Problems
Methodological perspectives provide frameworks for your agent. If you need the agent to act as a logic expert, you can apply the reasoning modules. These modules force the agent to follow specific steps before it gives you an answer.
